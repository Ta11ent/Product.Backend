# Online store
The backend part of the online store. The REST Api is an application implemented on a microservices architecture.

# Stack
- Asp Net Core
- MS SQL
- Rabbit MQ
- Redis
- MailKit

# Packages
- MediatR
- EntityFrameworkCore
- AutoMapper
- Fluent Validation
- Ocelot
- Swashbuckle
- MassTransit.RabbitMQ
- MailKit
- AspNetCore.Authentication
- JwtBearer
