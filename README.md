# Online store
The backend part of the online store. The REST Api is an application implemented on a microservices architecture.

# Stack
- Asp Net Core
- MS SQl
- Rabbit MQ

# Packages
- MediatR
- Entity Framework Core
- AutoMapper
- Fluent Validation
- Ocelot
- Swashbuckle
- RabbitMQ.Client
- MailKit
- AspNetCore.Authentication.JwtBearer
